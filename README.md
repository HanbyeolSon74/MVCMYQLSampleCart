### 2. 꾸미기 (마크다운 문법)
- **제목**: `#`을 사용하여 제목을 만들 수 있음 (`#`은 H1, `##`는 H2, `###`는 H3)
- **코드 블록**: 백틱(```)을 사용
- **링크**: `[링크 텍스트](URL)`
- **이미지 추가**: `![이미지 설명](이미지 URL)`
- **리스트**: `-` 또는 숫자를 사용 (`1.`, `2.`)
- **굵게/기울임**: `**굵게**`, `*기울임*`
