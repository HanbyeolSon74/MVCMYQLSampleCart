### Dayke
- **프로젝트 소개**: MVC, MYSQL을 사용하여 만들었습니다.
- **개발 동기**: NUDAKE를 좋아해서 만들게 되었습니다.
- **화면 구성**: 관리자 페이지, 메인 페이지, 가게 페이지, 메뉴 페이지, 상세 페이지
- **주요 기능**: 상품 등록, 상품 수정, 상품 삭제, 등록한 상품 페이지에 불러오는 기능을 구연했습니다.
